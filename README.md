# ROOM RESERVATION

This is my project for building room reservation scalable backend API.

## TECH STACK

- Node.js, Express
- Database: MongoDB
- Postman Collection
- Testing: Jest

## SETUP COMMANDS

```bash
git clone https://github.com/Amancodes26/Room-Reservation-Backend-2025.git
npm install
npm run dev
npm test
```

**Note:** Download Postman collection for API testing. Use reference of `.env.example` for setup.
